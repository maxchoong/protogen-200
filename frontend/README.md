# Film & TV Advisor Frontend

A responsive React + TypeScript web application for getting film and TV recommendations based on natural language input.

## Development

Install dependencies:
\`\`\`bash
npm install
\`\`\`

Run development server:
\`\`\`bash
npm run dev
\`\`\`

Build for production:
\`\`\`bash
npm run build
\`\`\`

Preview production build:
\`\`\`bash
npm run preview
\`\`\`

## Features

- Natural language input for "what I'm in the mood for"
- Optional preference questions (genres, mood, type, rating)
- Results page with recommendations
- Responsive design for mobile and desktop

## Architecture

- Uses Vite for fast development and optimized builds
- Component-based structure with Pages
- Tailwind CSS for styling
- TypeScript for type safety
- Proxy configured to backend API (localhost:5000)
